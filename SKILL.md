# 高质量女角色卡片 / 竖版海报提示词生成 Skill

## 1. Skill 目标

本 Skill 用于生成高质量的 **成年女性角色 3D CG 竖版卡片图 / 女角色竖版海报 / 角色宣传图 Prompt**。

适用场景包括：

- 即梦 AI / Seedream / Seedance 图像生成
- Midjourney、Stable Diffusion、可灵、通义万相等文生图模型
- 二次元 3D CG 女性角色图
- 幻想系女性角色海报
- 游戏角色宣传图
- 现代幻想时尚角色
- 甜美、清冷、梦幻、轻奢、星河、都市、卧室、窗边、海边等主题

本 Skill 的核心目的不是简单堆砌"美女""高质量""8K"等关键词，而是通过稳定的结构化框架，让模型明确理解：

1. 画面中是谁；
2. 用什么镜头看她；
3. 她处于什么动作状态；
4. 脸、皮肤、发型、服装分别如何表现；
5. 场景和光线如何服务主题；
6. 最终画面应该呈现什么渲染质感；
7. 哪些错误必须避免。

---

## 2. 核心框架

本 Skill 采用以下固定框架：

```text
主体定位
→ 构图镜头
→ 动作状态
→ 面部皮肤
→ 发型发色
→ 服装材质
→ 环境场景
→ 光线色彩
→ 风格渲染
→ 负面约束
```

建议生成 Prompt 时严格按照这个顺序组织。
这个顺序的好处是：

- 先锁定角色身份，避免模型跑题；
- 再锁定构图镜头，避免画面随机；
- 再描述动作状态，保证角色有姿态和生命感；
- 再细化面部、发型、服装，提高角色质量；
- 再补充场景、光线、色彩，提高画面完成度；
- 最后用风格渲染和负面约束兜底，减少崩图。

## 3. 使用原则

### 3.1 必须明确"成年角色"

所有女角色生成都必须明确为成年角色，避免幼态化、儿童感或未成年误判。

推荐固定写法：

```text
一位成年东方幻想系女性角色
一位成年二次元幻想系女性角色
一位成年现代幻想时尚女性角色
```

避免写法：

```text
少女、萝莉、幼态、童颜幼身、未成年感
```

---

### 3.2 气质要高级克制，不要低俗化

可以写：

```text
温柔、清冷、梦幻、甜美、优雅、慵懒、神秘、浪漫、治愈、知性、高级、空灵、自信、轻奢、软萌、疏离
```

避免写：

```text
低俗、挑逗、色情、过度暴露、媚俗、廉价网红感
```

---

### 3.3 身体描述以"姿态和线条"为主，不要过度堆身体部位

推荐写法：

```text
肩颈线条舒展，姿态自然优雅，腰部轻微收束，整体呈现成熟、健康、柔和、克制的女性美感。
```

谨慎写法：

```text
过度强调胸部、臀部、大腿、身体局部特写。
```

---

### 3.4 每个模块只写关键点，不要互相冲突

例如：

- 如果是"清冷星河女神"，就不要同时写"粉色女仆卧室"；
- 如果是"都市夜景轻奢"，就不要加入"古典音乐室、银河空间、海边建筑群"；
- 如果服装是"白色丝缎连衣裙"，就不要同时写"黑色皮革连体衣"。
