# Female Character Prompt Skill

一个用于生成高质量成年女性角色 3D CG / 二次元 / 幻想风 / 游戏角色 / 竖版海报提示词的结构化 Skill。

本 Skill 将女角色生图提示词拆解为一套稳定框架：

> 主体定位 → 构图镜头 → 动作状态 → 面部皮肤 → 发型发色 → 服装材质 → 环境场景 → 光线色彩 → 风格渲染 → 负面约束

它适合用于即梦 AI、Seedream / Seedance、Midjourney、Stable Diffusion、可灵、Runway、ComfyUI 等文生图或图生视频工作流中的提示词生成与改写。

---

## Features

- 结构化生成高质量女角色提示词
- 适合 9:16 竖版角色海报、人像卡片、幻想角色宣传图
- 内置主体、气质、构图、镜头、动作、发型、服装、材质、场景、光线、风格字段库
- 强调成年角色、高级克制、非低俗化表达
- 自动补充负面提示词，减少手部错误、肢体畸形、脸部崩坏、文字水印等问题
- 支持从简单主题扩写为完整高质量 Prompt

## Framework

本 Skill 使用以下固定框架：

```text
主体定位
→ 构图镜头
→ 动作状态
→ 面部皮肤
→ 发型发色
→ 服装材质
→ 环境场景
→ 光线色彩
→ 风格渲染
→ 负面约束
```

### 1. 主体定位

定义角色的基本身份、题材、年龄感、气质和整体审美方向。

示例：

```text
一位成年东方幻想系女性角色，整体气质为温柔、清冷、梦幻，具有精致二次元美型审美与高级 3D CG 角色质感。
```

### 2. 构图镜头

定义画面比例、人物位置、景别和镜头角度。

示例：

```text
9:16 竖版构图，人物位于画面中央偏左，构图为半身到大腿视角，镜头采用略高角度俯拍，主体清晰聚焦，背景柔和虚化。
```

### 3. 动作状态

定义人物姿态、身体方向、情绪状态和整体动作美感。

示例：

```text
人物身体微微前倾，肩颈线条舒展，腰部自然收束，姿态安静、柔和、优雅，整体呈现成熟、健康、克制的女性气质。
```

### 4. 面部皮肤

定义五官、眼睛、皮肤、妆容和表情细节。

示例：

```text
她拥有精致柔和的东方幻想系五官，白皙通透的肌肤，肤质细腻柔滑，带有柔和次表面散射质感、自然高光与淡淡红晕。
```

### 5. 发型发色

定义发色、发型结构、发丝动态和高光质感。

示例：

```text
发型为浅银蓝色中长发，发丝柔顺、轻盈、有层次，带有冷蓝色高光与透明感边缘光，部分发丝自然垂落在脸侧。
```

### 6. 服装材质

定义服装款式、材质、结构、配饰和整体造型完整度。

示例：

```text
服装为浅灰白色露肩针织毛衣，材质呈现柔软针织纹理，领口宽松自然，配饰包括黑色缎面颈带、蓝色晶体耳环与蝴蝶宝石项链。
```

### 7. 环境场景

定义角色所在场景、背景元素和主题氛围。

示例：

```text
场景为明亮的室内窗边，背景包含浅色窗帘、柔和室内环境和一杯咖啡，背景干净简洁，营造温暖、安静、治愈的氛围。
```

### 8. 光线色彩

定义光源方向、光影层次、主色调和画面氛围。

示例：

```text
清晨阳光从侧后方照入，形成柔和自然光、边缘轮廓光与浅景深光斑，整体色彩以浅蓝、奶油白、淡紫为主。
```

### 9. 风格渲染

定义整体渲染质量和美术风格。

示例：

```text
高质量 3D CG 角色渲染，半写实二次元美型审美，PBR 材质，电影级灯光，柔和次表面散射，高精度发丝渲染，真实布料纹理，浅景深，超清细节。
```

### 10. 负面约束

排除低质量、崩坏、低俗化和画面污染问题。

示例：

```text
不要水印，不要文字，不要 logo，不要低清晰度，不要脸部崩坏，不要眼睛不对称，不要手部畸形，不要多余手指，不要多余肢体，不要身体比例错误，不要背景杂乱，不要过度暴露，不要低俗色情化表达，不要幼态化。
```

## Install

使用 `npx skills` 安装：

```bash
npx skills@latest add hry1729/female-character-prompt-skill
```

如需使用本地目录安装：

```bash
npx skills@latest add ./female-character-prompt-skill
```

查看已安装 Skills：

```bash
npx skills@latest list
```

> 如果你的 GitHub 用户名或仓库名不同，请将 `hry1729/female-character-prompt-skill` 替换为你自己的仓库地址。

## Safety and Style Notes

This skill is designed for creating adult female character prompts with a refined, high-quality visual style.

Recommended constraints:

- The character should be clearly adult.
- The visual expression should remain elegant, premium, and restrained.
- Avoid infantilization.
- Avoid explicit or vulgar sexualization.
- Avoid excessive exposure.
- Avoid prompts that focus only on body parts.
- Prefer fashion, character design, portrait, fantasy, editorial, or game-character language.
